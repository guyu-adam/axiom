# Miser

> AI agent 开发与本地文件操作之间的桥梁。

## 是什么

Miser 是一个 AI agent 开发框架，专注于让 AI 安全、高效地操作本地文件系统。核心能力：
- 多 agent 适配（8 个后端）
- 安全沙箱层（防止 agent 越权操作）
- 500+ 测试用例

## 为什么存在

当前 AI agent 和本地文件系统之间的交互是脆弱的——缺乏安全层、缺乏标准化的操作接口、不同 agent 后端的适配成本高。Miser 解决这个 Gap。

## 为什么值得做（对你的意义）

- **代码杠杆的典型案例**：写一次，运行无数次。每条测试用例都是资产。
- **最快能产生被动收入的路径**：开源版引流 → 付费托管版
- **验证"我能把一件事从零推到上线"**：不只是造了，是推出去让人用了

## 当前状态

- 开发中，核心功能基本完成
- 待做：Landing page、社区推广、付费版搭建
- 目标：月入 $300-500
