# Noesis

> 用 LLM 做计算实验，桥接 Global Workspace Theory (GWT) 和 Integrated Information Theory (IIT)。

## 是什么

一篇学术论文。基本思路：意识的两大理论——GWT（意识是"全局工作空间"里的信息广播）和 IIT（意识是系统整合信息的量 Φ）——分别抓住了意识的不同侧面。用 LLM 作为计算模型，模拟两种理论预测的意识行为差异，寻找交汇点。

## 为什么存在

意识科学的两大理论阵营几乎不对话。GWT 偏向认知心理学和神经科学，IIT 偏向信息论和数学。Noesis 的核心赌注是：**LLM 可以充当一个"中立的计算实验平台"**，让两种理论的预测在同一个系统上进行对比测试。

## 为什么值得做（对你的意义）

- **学术 IP 的种子**：一篇论文 → 一个研究方向的入场券。你不需要成为职业学者——但有一个发表的、跨学科的、在意识科学领域的论文，你在这个话题上的发言权完全不同。
- **跨学科组合的极致体现**：天体物理 + AI 工程 + 意识科学。这个组合在全球范围内极少有人具备。
- **不可替代性**：Miser 在商业上可以被替代。Noesis 的独特视角——用计算模型验证意识理论——不太容易被复制。
