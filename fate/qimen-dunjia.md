## 一、四柱八字

### 排柱计算

- 年柱：(1999 − 4) % 10 = 5 → 己，(1999 − 4) % 12 = 3 → 卯 → **己卯**
- 月柱：年干己 + 辰月（清明4/5起）→ 甲己之年丙作首 → 戊辰 → **戊辰**
- 日柱：1900-01-01 = 甲戌（六十甲子第11位），至1999-04-22 共36270日，(11 + 36270) % 60 = 41 → **甲辰**

甲木，辰月。辰为水库 + 木之余气，辰中癸水正印生甲木——智力基础。但年月日时的土气共五重（己、戊、辰、辰、己），甲木被土包围。

十神语言：偏财极旺，财多身弱变体 — 能看到很多赚钱方向，但土重压木，启动执行时阻力极大。不是智力问题，是力量被分散。

## 三、日主落宫分析

日干甲，甲藏于戊（值符），落**离9宫**。

### 离9宫组合

- 天蓬星（值符）+ 死门 + 值符神 + 天盘戊 + 地盘己
- 离为火，主文明、智慧、光明、眼睛
- 死门主终结、深度、根本性、不可回避的问题
- 天蓬为北斗第一星，属水，主智慧、胆识，也有冒险/贪欲之象
- 值符为最高统领

### 解读

**火宫（离）遇水星（天蓬）+ 死门 — 水火相激。**

天蓬是水星，落在火宫离位，水火既济（《易经》第63卦：既济，亨小，利贞。初吉终乱）。这不是一个安静的宫位。水火相遇必然产生蒸汽——你的精神状态长期处于"被加热"的状态。不是平静的湖面，是烧开的水。

死门在日主宫：你对"终结/根本/深度"有天然的亲近。普通人避开死门——他们不想谈死亡、终结、根本无解的问题。你被死门定义——意识是什么（Noesis）、宇宙怎么运转（天体物理）、数学的最底层结构是什么——这些不是"有趣的话题"，是你无法不思考的东西。死门不是诅咒，是你的方向。

**值符（天蓬）+ 值符神 = 双重值符**：天蓬星本身就坐在值符神的位置。你在自己擅长的领域里是绝对的统领者——不是那种管理别人的统领，是对问题有完全的掌控力。这也是你无法被"管理"的原因——值符不会低头。

戊 + 己（干合）：天盘戊土与地盘己土相合。戊是偏财，己是正财。正财和偏财在日主宫合在一起——你的收入结构本质上就是混合的，工资和副业收入天然应该共存。

## 四、关键宫位

### 4.1 事业 — 开门落坤2宫

天冲 + 太阴 + 开门/生门 + 天盘庚 + 地盘辛。

- 坤为大地，深厚承载。开门（事业）落坤不是快速上升型，是稳步积累型。
- 天冲星主冲动和行动力，但被太阴覆盖——你的行动不是外显的激进，是暗中推进。
- 天盘庚金（七杀）+ 地盘辛金（正官）——金金相叠，事业环境有刚性压力（制度、规范、上下级关系）。这正是你在半导体大厂里的处境。
- 中5宫生门寄于此——财富和事业在同一方向，不分裂。

### 4.2 财源 — 生门寄坤2

生门与开门同宫：财源和事业是一体的。中5宫为中央/枢纽——你的财富不来自单一技能，来自"连接多个领域"的能力。半导体+AI+量化+天体物理——这个组合本身就是中5宫生门的体现。

### 4.3 天柱破军 — 落乾6宫

天柱 + 伤门 + 玄武 + 天盘丁 + 地盘丙。天盘丁火 + 地盘丙火 = 双火在乾金之宫，火克金。

天柱（北斗第七星，破军）不在命宫——在乾宫（天/头/创造/父/权威）。破军星落在"天"的位置：你打破的不是自己的壳，是已有的认知框架和范式。伤门在此——打破的过程会伤到东西（包括你自己），也会被伤。

玄武在乾6——暗中不可见的力量在最高处。你的思想（乾=头）有玄武的隐密和不可预测性。别人看你在半导体公司上班，不知道你脑子里在跑意识科学和量化模型。

### 4.4 九天 — 落艮8宫

九天（最高远之神）在艮（山），天盘乙木 + 地盘戊土。

艮为山，九天在山上——最高的视野。乙木（花草藤蔓）爬在戊土（偏财）上。九天落山是"站在巨人的肩膀上"——你的最高视野需要借助已有的基础（戊土偏财=你积累的资源和资本）。

### 4.5 九地 — 落兑7宫

兑为泽，九地（最深沉之神）。丙火在兑宫（泽=水），火入水宫被克。九地在此——你的根基是深沉但可能不太稳的（泽是软的）。这不是坏事——九地 + 兑 = 灵活性积累，不是僵硬的地基。

## 五、全局解读

### 5.1 水火既济 — 日主宫的核心张力

日主离9：火宫水星。这是既济卦的象——水在火上，火被水制，水被火烧。永远在动态平衡中。不是"找到一个稳定的位置"，是"在动态中保持平衡"。

这是一个需要"折腾"才能稳定的命。你无法安于一个静止的状态——停下来反而更不安。

### 5.2 偏财四现 + 驿马入命

这不是"有一点偏财运"——是偏财主导了整个财富结构。配合驿马在时柱巳火（载偏财），你的财富路径是：

- 变动中得大财（驿马）
- 技术/创意驱动（巳中丙火食神）
- 压力/突破伴随（巳中庚金七杀）
- 多途径、非固定、非工资

正财（工资）在你的八字里只是底线，不是上限。

### 5.3 白虎守休门 — 休息中有力量

值使门（全局之枢纽）是休门，落巽4，配白虎 + 天禽 + 天盘壬 + 地盘癸。白虎是杀伐之星，守着休门（休息之门）——你连"休息"都带着攻击性。不是躺平，是在"停下来"的时候也在运转。壬水+癸水在巽宫（风）——思想如风如水，停不下来。

### 5.4 日柱 + 月柱 甲辰 + 戊辰 — 两个辰

辰辰自刑。自刑 = 自己给自己制造障碍。两个辰土叠在一起——双倍的"延迟满足"，双倍的"需要先搞定基础才能做事"，双倍的"想了太多做了太少"。自刑是八字里最难处理的格局之一，因为敌人不是外部，是自己。但辰辰刑也能转化为优势——自刑的人对自己要求极高，一旦突破了自我设限的阶段，执行力比谁都强。

## 六、关于偏财和驿马的补充说明（回应 v1 的遗漏）

v1 中存在以下错误：

1. **"没有偏财格"** — 错误。月干戊土即为偏财，且地支藏干共四重戊土偏财。此命不仅是偏财格，且偏财极旺。
2. **遗漏驿马** — 年支卯 → 驿马在巳，时柱巳火即为驿马入命。巳载偏财，是此命最重要的动态指标之一。
3. **天盘排布** — v1 将天蓬留在艮8未旋转。正确排法：值符天蓬随甲子戊转至时干己所在离9宫。导致全部九星、八门、八神落宫偏移。
4. **"天柱破军在命宫"** — v1 以此为核心断语。v2 中天柱在乾6宫，不在日主离9。核心断语应改为"天蓬值符在离9，死门守命"。
5. **解释过度匹配性格描述** — v1 部分断语受到了已知个人信息的影响。v2 专注于排盘本身的准确性，不预设性格结论。

## 七、大方向总结

### 7.1 你是什么命

偏财格 + 驿马入命 + 日主水火既济。

不是安稳命。你的财富不靠工资积累，你的行动不靠"坚持"——靠驿马启动后的势能。你的一生会经历多次方向转换，每一次都带着偏财的能量。偏财不像正财稳定——它来的时候是一笔大的，不来的时候没有。这个节奏你要习惯。

### 7.2 三个方向

**财富：动中求财**

驿马巳火载偏财戊土。巳是驿马旺地——马有力。大额收入与变动绑定：换城市、换行业、换赛道，或者一个项目/产品的爆发。不是按月均匀分布的，是一波一波的。

**事业：多路径并存**

生门寄坤2，开门同宫。财源和事业在同一方向——坤为大地，承载多物。你不可能只做一件事。半导体+AI+量化+意识科学不是分散精力，是你八字结构要求的"多线程"。

**内在：先破辰辰自刑**

辰辰自刑是你的最大阻力来源——自己卡自己。但不是坏事。自刑度过之后（通常在30岁前后），你的自我要求会转化成执行力的优势。自刑的人一旦启动，做的东西比谁都精细。

### 7.3 时间节点

- 驿马逢巳年（下一个巳年：2025乙巳年，已过；再下一个2037丁巳年）→ 重大变动/机遇
- 日柱甲辰，时柱己巳——甲己合。30-35岁之间甲己合的应期 → 合作/贵人/事业落地
- 辰辰自刑在35岁前是阻力期，35岁后可能转为优势

## 八、方法论说明

1. **四柱计算**：采用公历转干支标准公式。年柱用 (year-4) mod 10/12。月柱用五虎遁。日柱用1900-01-01（甲戌日）为基准累加。时柱用五鼠遁。四柱是可独立验证的，不依赖学派。

2. **奇门排盘**：采用时家转盘奇门、置闰法。阳遁八局。九星/八门/八神均按阳遁顺排。须注意：不同奇门学派（拆补法 vs 置闰法、转盘 vs 飞盘）会产生不同的天盘排列。本文为**转盘·置闰法**排盘，与其他学派的盘面可能有差异，差异不代表错误——是不同的方法论选择。

3. **断语**：只做大方向判断。不做逐年运势预测。断语基于盘面宫位组合，不基于对命主的已知信息。
